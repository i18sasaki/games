# 作品名:Capricciosopus

**作品概要**

リズムに合わせてボタンを押すことで高いスコアを目指すゲームです。
UI,3Dオブジェクト,C#スクリプトをすべて自作しました。
アカウント名、パスワードと最高スコアをjsonファイルで一括管理することでスコアを保存したりできるようにしました。
